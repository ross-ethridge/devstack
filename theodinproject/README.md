# The Odin Project

Solutions to the problems on 'the odin project' ruby courses.

## Ceaser Cipher

```text
./ceaser_cipher.rb --help
Usage: caesar_cipher.rb [options]
    -s, --string STRING              The string to be encrypted
    -n, --shift SHIFT                The shift value for the cipher, use negative values for decryption


./ceaser_cipher.rb -s "Hello World" -n 5
Mjqqt Btwqi


./ceaser_cipher.rb -s "Mjqqt Btwqi" -n -5
Hello World
```
